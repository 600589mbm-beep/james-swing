# Publishing this repo to GitHub (no coding required)

You don't need the command line. The easiest path uses **GitHub Desktop** (a free app with buttons).

## One-time setup (~10 min)

1. **Create a free GitHub account** at https://github.com (use a professional username — recruiters will see it).
2. **Download GitHub Desktop** from https://desktop.github.com and install it.
3. Open GitHub Desktop and **sign in** with your new account.

## First: delete the leftover `.git` folder

This folder may contain a hidden, incomplete `.git` subfolder. **Delete it before importing** so GitHub Desktop can make a clean one:
- In File Explorer, open the `2ndswing-ai-toolkit` folder, click **View → Show → Hidden items**, then delete the `.git` folder if you see one. (Your real files — `index.html`, `demos`, `docs`, etc. — stay untouched.)

## Publish the project

4. In GitHub Desktop: **File → Add Local Repository**, then choose this `2ndswing-ai-toolkit` folder. When it says the folder isn't a repository, click **"create a repository"**.
   - If it says "this isn't a Git repository," click **"create a repository"** when prompted (the app handles it).
5. You'll see all the files listed as changes. In the bottom-left box, type a summary like `Initial commit — AI toolkit & ROI proposal` and click **Commit to main**.
6. Click **Publish repository** (top bar).
   - **Uncheck "Keep this code private"** if you want the hiring manager to see it without being invited. (For a portfolio piece, public is usually better.)
   - Click **Publish**.
7. Done — click **View on GitHub** to get your link. It'll look like `https://github.com/yourname/2ndswing-ai-toolkit`.

## Make it look sharp (5 min)

- On the repo page, click the ⚙️ next to **About** and add a one-line description + the demo note.
- GitHub auto-displays your `README.md` on the front page — it's already written to impress.
- **Optional but powerful:** turn on **GitHub Pages** (Settings → Pages → Source: `main` / root) so your demos open as live links anyone can click — no download needed. Your listing agent would live at `https://yourname.github.io/2ndswing-ai-toolkit/demos/listing-agent.html`.

## What to send

Once it's live, the repo link + a 2-minute screen recording is your application. Lead with the demo, not the résumé.
